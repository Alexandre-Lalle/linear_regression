Regression Module¶

The regression module provides high-level access to its components, including linear regression modeling and grid search.

class LinearRegression(learning_rate: float = 0.001, max_iter: int = 1000, tol: float = 0.0001, l1_ratio: float = 0.0, l2_ratio: float = 0.0, normalize: bool = False, remove_outliers: bool = False)[source]¶

Bases: object

A multiple linear regression model implementing gradient descent optimization.

Parameters:

  • learning_rate (float, default=0.001) – The learning rate for gradient descent

  • max_iter (int, default=1000) – Maximum number of iterations for gradient descent

  • tol (float, default=1e-4) – Tolerance for the stopping criterion

  • l1_ratio (float, default=0.0) – L1 regularization strength (Lasso)

  • l2_ratio (float, default=0.0) – L2 regularization strength (Ridge)

  • normalize (bool, default=False) – Whether to normalize features before fitting

  • remove_outliers (bool, default=False) – Whether to remove outliers before fitting

Returns:

y_pred – Predicted values

Return type:

array-like of shape (n_samples,)

Warning

If normalize is set to False during fit, this method does not automatically apply feature normalization to the input X. Ensure that the features of X match the scale used during training to avoid inconsistent predictions. If normalization was applied during training, X will be normalized automatically here.

Perform a grid search to find the best hyperparameters for LinearRegression.

Parameters:
