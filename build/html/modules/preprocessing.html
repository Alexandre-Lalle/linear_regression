Data Preprocessing¶

The preprocessing submodule offers utility functions for data normalization, cleaning, and splitting.

normalize_features(X: ndarray) → ndarray[source]¶

Normalize features by removing the mean and scaling to unit variance.

Parameters:

X (np.ndarray) – The input features to be normalized.

Returns:

The normalized features.

Return type:

np.ndarray

remove_outliers(X: ndarray, y: ndarray) → Tuple[ndarray, ndarray][source]¶

Remove outliers based on the IQR (Interquartile Range) method.

Parameters:

  • X (array-like of shape (n_samples, n_features))

  • y (array-like of shape (n_samples,))

Returns:

X_clean, y_clean

Return type:

Tuple of arrays without outliers

train_test_split(X: ndarray, y: ndarray, test_size: float = 0.2, random_state: int | None = None) → tuple[source]¶

Split arrays or matrices into random train and test subsets.

Parameters:

  • X (np.ndarray) – The input features.

  • y (np.ndarray) – The target values.

  • test_size (float, optional) – The proportion of the dataset to include in the test split (default is 0.2).

  • random_state (int, optional) – Controls the shuffling applied to the data before applying the split (default is None).

Returns:

A tuple containing the train-test split of inputs and targets: (X_train, X_test, y_train, y_test).

Return type:

tuple
